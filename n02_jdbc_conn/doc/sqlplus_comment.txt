=================== commit 에 관하여 ==============================
1. 프로그램상에서 sql (insert)문을 작성하고 실행한 것은 --- 자동 commit 된다.
   콘솔창에서 sql> insert into ~~  실행한것은 ---- 자동 commit 안된다.
           sql> commit; //이부분을 해줘야 데이타가 반영된다.
2. DDL 문장은 자동 commit 된다
   DML 문장은 자동 commit 안된다..별도로 해줘야 한다.
